# Complaint Classifier

## 📌 Project Description
This project is a Complaint Classifier developed using Python. It classifies user complaints into different categories based on the input data.

## ⚙️ Technologies Used
- Python
- CSV Files

## 📂 Files in Project
- classifier.py → Main Python code for classification
- input.csv → Input data (complaints)
- output.csv → Output data (classified complaints)

## 🚀 How it Works
1. The user provides complaint data in input.csv
2. The Python script processes the data
3. The complaints are categorized
4. Results are stored in output.csv

## 🎯 Features
- Automatic complaint classification
- Easy to use
- Fast processing
